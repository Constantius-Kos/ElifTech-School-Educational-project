# Delivery App 🍔🚚

**🌐 [Live Demo (Демо проекта)](https://delivery-frontend-z69q.onrender.com)**

Тестовое задание / Учебный проект для **ElifTech School** по созданию полноценного веб-сервиса доставки еды и товаров.

## 🛠 Технологический стек



*   **Фронтенд:** React, TypeScript, Vite, CSS Modules
*   **Бэкенд:** Node.js, Express, TypeScript
*   **База данных:** MongoDB (Atlas) + Mongoose
*   **Архитектура:** Monorepo (папки `packages/frontend` и `packages/backend`)

---

## 🚀 Реализованный функционал (по требованиям ТЗ)

Проект разбит на три уровня сложности:

### 🥉 Базовый уровень (Base Level)
*   **Бэкенд:** Реализовано собственное REST API (Express + MongoDB).
*   **Страница "Shops":**
    *   Сайдбар со списком доступных магазинов.
    *   При выборе магазина подгружается список товаров исключительно этого магазина.
    *   Кнопка "Add to Cart" на карточках товаров.
*   **Страница "Shopping Cart":**
    *   Форма для оформления заказа (Имя, Email, Телефон, Адрес доставки).
    *   Список выбранных товаров с возможностью изменить количество (инпуты) или удалить позицию.
    *   Подсчёт итоговой суммы (Total Price).
    *   Сохранение заказа в базу данных при сабмите.

### 🥈 Средний уровень (Middle Level)
*   **Отзывчивый дизайн (Responsive):** Адаптация интерфейса под мобильные устройства. Меню и карточки продуктов перестраиваются для удобного использования со смартфона.
*   **Сортировка (Sorting):** Возможность сортировки меню по цене (Price) и алфавиту (A-z/Z-a).
*   **Фильтрация (Filtering):** Фильтр товаров внутри магазина по уникальным категориям (Drinks, Burgers, Consoles и т.д.) вместо общего списка.

### 🥇 Продвинутый уровень (Advanced Level - *В разработке*)
*   *Пагинация / Бесконечный скролл списка товаров.*
*   *Страница Истории заказов (Order History) с поиском по Email / Телефону.*
*   *Функция "Reorder" — быстрое добавление старого заказа в корзину.*
*   *Страница купонов и промокодов.*
