# Delivery App 🍔🚚

Тестове завдання / Навчальний проєкт для **ElifTech School** зі створення повноцінного вебсервісу доставки їжі та товарів.

## 🛠 Технологічний стек

*   **Фронтенд:** React, TypeScript, Vite, CSS Modules
*   **Бекенд:** Node.js, Express, TypeScript
*   **База даних:** MongoDB (Atlas) + Mongoose
*   **Архітектура:** Monorepo (папки `packages/frontend` та `packages/backend`)

---

## 🚀 Реалізований функціонал (за вимогами ТЗ)

Проєкт поділено на три рівні складності:

### 🥉 Базовий рівень (Base Level)
*   **Бекенд:** Реалізовано власне REST API (Express + MongoDB).
*   **Сторінка "Shops":**
    *   Сайдбар зі списком доступних магазинів.
    *   При виборі магазину завантажується список товарів виключно цього магазину.
    *   Кнопка "Add to Cart" на картках товарів.
*   **Сторінка "Shopping Cart":**
    *   Форма для оформлення замовлення (Ім'я, Email, Телефон, Адреса доставки).
    *   Список обраних товарів із можливістю змінити кількість (інпути) або видалити позицію.
    *   Підрахунок підсумкової суми (Total Price).
    *   Збереження замовлення в базу даних при сабміті.

### 🥈 Середній рівень (Middle Level)
*   **Адаптивний дизайн (Responsive):** Адаптація інтерфейсу під мобільні пристрої. Меню та картки продуктів перелаштовуються для зручного використання зі смартфона.
*   **Сортування (Sorting):** Можливість сортування меню за ціною (Price) та алфавітом (A-z/Z-a).
*   **Фільтрація (Filtering):** Фільтр товарів усередині магазину за унікальними категоріями (Drinks, Burgers, Consoles тощо) замість загального списку.

### 🥇 Просунутий рівень (Advanced Level - *В розробці*)
*   *Пагінація / Нескінченний скрол списку товарів.*
*   *Сторінка Історії замовлень (Order History) із пошуком за Email / Телефоном.*
*   *Функція "Reorder" — швидке додавання старого замовлення до кошика.*
*   *Сторінка купонів та промокодів.*
